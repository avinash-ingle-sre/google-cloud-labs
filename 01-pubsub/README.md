# 📨 Google Cloud Pub/Sub

## 📌 Context
Pub/Sub is Google Cloud's async messaging service based on Publisher/Subscriber pattern.

## 🎯 Use Case
- Microservices को decouple करणे
- Event-driven architecture
- Real-time data streaming (IoT, logs, analytics)

## 💥 Impact
- Services independently scale होतात
- No message loss — guaranteed delivery
- Millions of messages/sec handle करू शकतो

## 🔑 Key Concepts
| Term | Meaning |
|------|---------|
| Topic | Message channel/queue |
| Subscription | Message receiver |
| Publisher | Message पाठवणारा |
| Subscriber | Message घेणारा |
| auto-ack | Read केल्यावर message delete |
